## 📸 Preview

A premium, responsive personal portfolio showcasing professional experience, education, technical skills, certifications, and projects — designed with an exclusive tech-inspired aesthetic using a deep navy and electric blue palette.

---

## ✨ Features

- **Responsive Design** — Mobile-first layout with breakpoints at 1024px, 768px, and 480px
- **Animated Hero** — Typewriter effect showcasing multiple roles
- **Work Experience Timeline** — Chronological career history with official company logos linked to their websites
- **Education Section** — Academic background from SMSU (MS), IUB (BS), and pre-university credentials
- **Interactive Skills Grid** — Categorized technical competencies
- **Project Showcase** — Featured work with technology tags
- **Certifications Grid** — Professional certifications and training
- **Contact Section** — Direct links to email, LinkedIn, and GitHub
- **CV Download** — One-click resume access
- **All logos are externally hyperlinked** to their respective official websites

Any push to `main` automatically rebuilds and redeploys the live site within ~60 seconds.
